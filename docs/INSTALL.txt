dgcn.content Installation
-------------------------

To install dgcn.content using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``dgcn.content`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        dgcn.content
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
